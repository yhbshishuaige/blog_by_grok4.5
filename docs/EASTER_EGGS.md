# 彩蛋维护手册

本文档用于维护、调试和验收，不面向普通访客。触发逻辑以 `js/secrets.js` 为准，视觉效果主要位于 `styles/secrets.css`。

## 状态与规则

- 收集进度：`localStorage["weather-blog-secrets"]`
- 星尘开关：`localStorage["weather-blog-stardust"]`
- 收集总数：14；首次成功触发才计入进度。
- 重型场景互斥：极光、流星雨、云上星鲸不会同时播放。
- 减少动态效果或 `hardwareConcurrency <= 4` 时，部分临时粒子数量会降低。
- 页面切到后台后，装饰动画统一暂停。

## 14 个可收集彩蛋

所有彩蛋都可在控制台用 `WeatherBlog.secrets.preview("ID")` 预览；预览仍可能受背景、天气等内部条件影响，表中另有说明。

| ID | 名称 | 触发条件与操作 | 触发效果 | 时长 |
|---|---|---|---|---|
| `aurora` | 极光 | 约 1.7 秒内连续点击左上角圆形 Logo 5 次；大雨、雷电时拒绝 | 天空出现极光，并清除其他重型场景 | 11 秒 |
| `meteors` | 流星雨 | 夜间 19:12–次日 05:12 点击月亮；白天只显示提示 | 生成 13 颗流星，低性能设备为 7 颗 | 5.2 秒 |
| `fireflies` | 山脚萤火 | 时间处于 18:30–20:30，点击时钟 | 山脚生成 26 只萤火虫，低性能设备为 12 只 | 12 秒 |
| `blue-hour` | 蓝调时刻 | 时间处于 04:48–06:18，点击时钟 | 页面进入短暂的蓝调氛围 | 9 秒 |
| `rainbow` | 雨后彩虹 | 手动把任意雨天、雷电或大风切换为晴天 | 约 0.52 秒后出现雨后彩虹 | 9 秒 |
| `constellation` | 星鲸座 | 夜间点击星空；白天只显示提示 | 星星连成鲸形星座 | 10 秒 |
| `snow-fox` | 雪狐 | 选择雪山背景并切到任意雪天，长按天气图标约 0.52 秒；累计唤醒 3 次才发现 | 山脊雪狐醒来，第 3 次进入高兴状态 | 每次 4.8 秒 |
| `moon-halo` | 零点月晕 | 用时间轮盘进入手动预览并调到 00:00–00:01 | 月亮周围出现月晕 | 8.5 秒 |
| `prism` | 12:34 光棱 | 用时间轮盘进入手动预览并调到 12:33–12:35 | 天空出现短暂光棱 | 6.5 秒 |
| `sky-whale` | 云上星鲸 | 页面非输入状态下依次键入 `sky`；大雨、雷电时拒绝 | 星鲸穿越云层，并清除其他重型场景 | 13 秒 |
| `sun-halo` | 太阳日晕 | 白天 05:12–19:12 点击太阳；夜间只显示提示 | 太阳周围出现暖色日晕 | 6.2 秒 |
| `paper-boat` | 雨中纸船 | 任意雨天或雷电天气点击页脚 | 纸船沿页面底部漂过 | 9.2 秒 |
| `dandelion` | 阴天蒲公英 | 阴天天气点击页脚 | 蒲公英种子随风飘过 | 10.5 秒 |
| `echo` | 山谷回声 | 雪山背景、非大风天气，点击峰顶附近隐藏热点；悬停约 0.8 秒会出现极淡提示 | 山谷产生回声视觉回应 | 4.2 秒 |

补充：点击时钟但不在蓝调或萤火时间段，只会提示隐藏景色出现的时间线索，不计入收集。

## 发现手册与全收集奖励

- 桌面端右键右上角“背景”按钮，可查看已发现列表。
- 触屏设备长按“背景”按钮约 0.72 秒，效果相同。
- 14/14 后自动解锁低密度星尘模式。
- 全收集后再次打开发现手册，会在开启和关闭星尘模式之间切换。
- 背景按钮会通过样式标记“已有发现”和“已全部完成”状态。

## 调试 API

```js
WeatherBlog.secrets.list();                  // 返回全部可收集 ID
WeatherBlog.secrets.preview("aurora");       // 预览指定彩蛋
WeatherBlog.secrets.tour();                  // 巡演极光、流星、星鲸、彩虹、星鲸座
WeatherBlog.secrets.stopTour();              // 停止巡演和主要临时场景
WeatherBlog.secrets.getDiscoveries();        // 返回已发现 ID
WeatherBlog.secrets.resetDiscoveries();      // 清空进度和星尘状态
```

逐项验收时建议先用 `WeatherBlog.secrets.preview(ID)` 检查画面，再按表格中的真实交互检查前置条件、提示、收集计数和持续时间。
